## Fleet Management System

A console-based fleet management application built in Java. 
This project demonstrates OOP fundamentals by modeling cars and vehicles with attributes like speed, location, and license plate, plus actions like drive, stop, and setSpeed.

## Features
- **Create cars**: Initialize vehicles with brand, year, model, transmission, and license plate
- **Track vehicle data**: Get brand, model, year, current speed, max speed, and location  
- **Control actions**: `drive()`, `stop()`, `setSpeed()`, `setLocation()`
- **Fleet tracking**: Count total cars created using `carCount`

## Tech Stack
- **Language**: Java 17+
- **OOP Concepts**: Classes, Objects, Constructors, Inheritance, Encapsulation, Getters/Setters
- **IDE**: IntelliJ IDEA
